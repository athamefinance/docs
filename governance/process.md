# Process

All preliminary discussions will happen in [Discord](https://discord.gg/C7FFvpry4p).

* The purpose of the Request for Channel is to establish formal discussion around a potential proposal.  In the Governance category in the requests channel please request a new channel to discuss the proposal with the following [information](template.md)
* Facilitate preliminary discussion
* Update and refine request to become a Proposal
* [Snapshot vote](https://vote.athame.finance)
